# SumLab - Personal Project Showcase!
 - This is a summative Lab done to showcase the REACT skills learnt through module 3.
 - Main files are all located within the src folder and the components folder within it.
# To set up:
  - Install node_modules through the command npm install
  - Git clone this repository onto your own computer into a selected file
  - Run npm run server to receive the database (db.json) and in a different terminal, run
    the command npm run dev to access the webpage.
